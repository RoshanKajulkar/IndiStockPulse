# IndiStockPulse

IndiStockPulse is a Java Spring Boot-based RESTful API providing historical stock market data for Indian markets. Built with robust JWT authentication, rate limiting, and PostgreSQL integration,request logging, caching, and exception handling for a scalable and high-performance API.
