IndiStockPulse

India's first independent, non-broker, free stock market data provider

https://indistockpulse.onrender.com/api/stocks/symbols [GET]

The API provides list of available symbols.

Response Attribute

Attribute Data Type Description
s
string
 ok / error
symbols
array
 Contains list of symbols
https://indistockpulse.onrender.com/api/stocks/data [POST]

The API provides 15min historical stock data from 2022 to 2024 Sept.

Request Attribute

Attribute Data Type Description
symbol*
string
 Mandatory. Eg: RELIANCE (list of available symbols can be fetched using /symbols api)
range_from*
string
 Accepts yyyy-mm-dd format
range_to*
string
 Accepts yyyy-mm-dd format

Response Attribute

Attribute Data Type Description
s
string
 ok / error
candles
array
 Candles data containing array of following data for particular time stamp:
1.Current epoch time
2. Open Value
3.Highest Value
4.Lowest Value
5.Close Value
